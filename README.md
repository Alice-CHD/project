本项目针对“双碳”目标下，未来高速公路下匝道因车辆电动化、智能化形成异质交通流所带来的“高碳排、高风险”痛点，提出一个名为“碳-安协同”的动态管控数字孪生系统。本系统创新性地构建了一个集“实时感知-仿真推演-智能决策-动态控制”于一体的闭环。
核心方案是利用车路协同（V2X）、毫米波雷达等多源数据，构建下匝道影响区的交通数字孪生体，实时计算碳排放与交通冲突风险。基于此，一个以深度强化学习为核心的AI决策引擎，能够根据实时的智能车/电动车混入率、交通密度等，动态优化并执行最佳管控策略组合（如：动态启用/关闭智能车专用道、发布可变速度引导、激励最优车队编组）。
项目创新价值在于，将静态的交通策略研究提升为动态、自适应的智能控制系统，实现了从“事后分析”到“事前预测、事中控制”的飞跃，能够显著提升通行安全、降低交通碳排放，为未来智慧高速的运营管理提供了前瞻性的解决方案。
